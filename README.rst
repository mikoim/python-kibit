Kibit Wrapper for Python3
=========================

This is a library for python to access Kibiti API.

Getting Started
===============

Please do the following::

  $ git clone https://github.com/mikoim/python-kibit
  $ cd python-kibit
  $ pydoc3 KIBIT
  $ less example.py
  $ python3 KIBIT/__init__.py # test
  $ python3 example.py
